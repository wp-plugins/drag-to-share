== Description ==

"Drag to share" is the newest trend in social sharing. Just drag an image and drop it into any social website to share the whole page in real time. This plugin is basically a similar effect release by Meebo.com and used by big players like Mashable. 

The plugin was developed on the source files created by Dan Wellman. Read more here: http://net.tutsplus.com/tutorials/javascript-ajax/drag-to-share
The credits for some of the design elements of this plugin goes to Bogdan Pop: http://www.bogdan-pop.com

The plugin complies to GNU General Public License, except the mascot which is copyrighted!

For more about this plugin visit http://www.adrian-apan.com/blog/wordpress-plugin-drag-to-share

== Installation ==

1. UnZip and Upload 'drag-to-share.zip' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place the image which you want to have the "Drag and Share" effect between &lt;div class=&quot;ui-draggable&quot;&gt; and &lt;/div&gt;
